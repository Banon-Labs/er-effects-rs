# ER Bevy Shader Lab

Small Bevy sandbox for iterating on WGSL shader ideas around Elden Ring-style visual effects without touching the active repo worktree.

Run from this worktree:

```bash
cargo run -p er-bevy-shader-lab
```

Tinker with `assets/shaders/elden_rune_lab.wgsl`; Bevy watches asset files and reloads the shader while the app is running.
